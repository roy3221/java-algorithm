File format for Knapsack problem inputs:

First row contains the total weight (W).
Following rows contain items in the following format:
Item_id Item_Weight Item_Value

---------------------------------
Example:
8
1 3 40
2 2 35
3 5 30
---------------------------------


Your program should print out at least the following information:
Value of your solution (OPT(W)) 
List of items that gives optimal value

---------------------------------
Example:
140
2 2 2 2
---------------------------------

OPTIONAL:
Print your data structure that stores the optimal values (e.g. array or 2D-array).
Print solution to a file.
